                              CORE API ENDPOINTS (LIVE)
================================================================================

🔐 AUTHENTICATION & USER MANAGEMENT
├── POST /auth/login                    - User authentication
├── POST /auth/signup                   - User registration
├── GET  /auth/verify                   - JWT token verification
└── Integrated with: Stripe, JWT, bcrypt

💳 PAYMENT & SUBSCRIPTION SYSTEM
├── POST /payments/create-checkout      - Stripe checkout session creation
├── POST /webhooks/stripe              - Stripe webhook handler
├── Pricing Plans: Free ($0), Standard ($10), Premium ($15), Enterprise ($25+)
└── Integrated with: Stripe Live Keys, Webhook Security

💬 AI CHAT & CONVERSATION SYSTEM
├── POST /chat                         - Multi-model AI chat (OpenAI, Anthropic, Google)
├── POST /conversations                - Create new conversation
├── GET  /conversations                - List user conversations
├── GET  /conversations/{id}/messages  - Get conversation history
└── Supports: GPT-4, Claude, Gemini with plan-based restrictions

🧠 MEMORY MANAGEMENT SYSTEM
├── POST /memory                       - Save to user memory/knowledge base
├── PUT  /memory/{key}                 - Update memory entry
├── GET  /memory                       - Retrieve all user memory
├── DELETE /memory/{key}               - Delete memory entry
├── GET  /memory/usage                 - Memory quota and usage stats
└── Features: Memory limits by plan, usage tracking, categories

🔍 SEARCH & DISCOVERY
├── POST /search                       - SerpAPI-powered web search
└── Integrated with: SerpAPI, search history tracking

🏥 SYSTEM HEALTH & MONITORING
├── GET  /                            - Root endpoint with feature status
├── GET  /health                      - Health check endpoint
├── GET  /models                      - Available AI models
└── Features: Database status, API availability, model routing
